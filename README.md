## 🎯 What It Does Quickly

The workbench answers two core questions instantly:

1. **"Where do I start?"** Pick any item across five frameworks (NIST, CSA, ISO, MITRE, OWASP) and instantly see everything connected to it.
2. **"What is my risk posture?"** Run a rapid, structured AI risk assessment and export a formatted executive report.

---

## 🚀 Getting Started (Zero Setup)

The application is built with standard web technologies. There is no server, no `npm install`, and no complex build pipelines required.

```bash
git clone https://github.com/Joieux/ai-governance-workbench.git
cd ai-governance-workbench
```

Open `ai-workbench-enterprise.html` directly in any modern browser:
- **macOS:** `open ai-workbench-enterprise.html`
- **Windows:** `start ai-workbench-enterprise.html`
- **Linux:** `xdg-open ai-workbench-enterprise.html`

---

## 🛠️ The Four Core Modes

1. **🔍 Explore:** A five-column bidirectional cross-reference map. Click any framework item to highlight connections instantly.
2. **📋 Assess:** Track your organization's implementation status (Done / In Progress / Not Started) across all 40 controls globally.
3. **📤 Export:** Generate clean markdown Gap Reports, Selection Mappings, and Control Inventories.
4. **🔍 Audit:** An end-to-end AI risk assessment workflow that profiles your system, suggests risk scenarios, and generates formal findings.
